# Mysura Reddy Kuchuru — Data & AI Portfolio

Source for a responsive, dependency-free terminal portfolio focused on data engineering, analytics engineering, data analysis, AI, and ML systems.

## Experience

- Terminal résumé structure with a persistent table of contents
- Cinematic, skippable terminal boot sequence with repeat-visit detection
- Premium dark and light themes
- Filterable project index and keyboard command palette
- Accessible navigation and reduced-motion support
- Print-ready HTML résumé
- Clear separation between completed work and future AI/ML roadmaps

## Preview locally

```bash
python -m http.server 8000
```

Open `http://localhost:8000`.
